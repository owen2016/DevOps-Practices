# Typora自动上传到Picgo图床

## Typora





## 配置

打开typora，点开左上角文件，选择**偏好设置**

1. 设置插入图片时为【上传图片】
2. 勾选【对本地位置的图片应用上述规则】
3. 在上传服务中选择“PicGo(app)”
4. 在路径中选择picgo安装目录 **PicGo.exe**

![20200919004945](https://gitee.com/owen2016/pic-hub/raw/master/pics/20200919004945.png)

点击下面的验证图片上传选项，上传成功！

上传的方法也很简单，将图片复制进去typora就会自动帮你上传了，你也可以右键点击上传图片
